## Summary

The Official website of StaffRex

## Tools & Libraries used

- NextJS
- React
- JavaScript

## Project Setup

Install dependencies and devdependencies, run:

using npm

```
npm install
```

using yarn

```
yarn install
```

## Running the app

To get the app up and running, run:

```shell
npm run dev
or
yarn dev
```

## Building the app

To get the app up and running, run:

```shell
npm run build
or
yarn build
```
